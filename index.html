How To Play

 

1. Make sure to allow pop ups, if prompted!

 

2. To play, just enter your channel name, then press "Start Game" button.

 

3. You don't have to reset the game if you change any of the settings, except the channel name.

 

4. There is no need to add a prefix like "!word" to make a guess.

 

5. The word can be anywhere in a message. Unless "Detect First Words Only" is checked
